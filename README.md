## Required work

### Main tasks

- Pseudo-Pascal syntax analyzer
- Pseudo-Pascal semantic analyzer
- Pseudo-Pascal interpreter
- Pseudo-Pascal -> C3A compiler
- C3A interpreter

### Optional tasks

- C3A -> Y86 compiler
- Enhance memory usage of the Y86 program made with the compiler
